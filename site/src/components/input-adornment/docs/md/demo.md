### 文本装饰

在输入框前后加入一些特定的纯展示标签，通常在需要提高辨识效率时使用。

<script>
import Text from '../../example/Text.svelte'
</script>

<Text></Text>

### 下拉列表装饰

装饰内容可以是下拉菜单。

<script>
import Select from '../../example/Select.svelte'
</script>

<Select></Select>

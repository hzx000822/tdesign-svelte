TDesign 适配桌面端的组件库，适合在 svelte 技术栈项目中使用。

# 🎉 特性

- 适配桌面端交互

# 📦 安装

```shell
npm i tdesign-svelte
```

# 🔨 基础使用

```js
import { TButton } from "tdesign-svelte";
```
